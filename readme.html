Local Development Setup Instructions

This is a work in progress. Please read carefully.

Let's get straight to it:

1. Install Virtual Box:

https://www.virtualbox.org/wiki/Downloads

2. Install Vagrant:

http://www.vagrantup.com/downloads.html

3. Create some directories

Make two a new directories on your machine, wherever you like. Read below for more info on what they are for. I have mine in /Users/raykamille/ 
local_development/
local_development/virtual-web-sites/
local_development/ is where you will clone the git repo for the virtual web server. local_development/virtual-web-sites/ is where you will clone git repo's for the various sites So put them some place accordingly.

4. Clone this git repo

into local_development/ #put the git repo here

5. Let's launch the new enviornment

Command line for now as this is version .01 
cd /path/to/local_development/vagrant/
vagrant up

6. While it's doing it's thing, add this to your 'hosts' file.

# Local Development Hosts
192.168.77.10 localhost.default

7. Wait for it to finish, the first run can take a little bit

8. Once it's done... you should be able to click this link and see a default wordpress install

http://localhost.default/ admin user - localdev/abc123  

Some notes for reference:

Vagrant is a wrapper for Virtual Box

Virtual Box is virtual machine software -vagrant is using it to a virtual machine on your computer

The virutal machine is running linux - debian squeeze to be more specific

The virtual machine is running a web server at 192.168.77.10

The virtual machine has a shared folder with your machine localdevelopment/virtual-web-sites/

localhost.default is served from here:

localdevelopment/virtual-web-sites/default/webroot
